Support¶

There are many ways to get in touch.

File an Issue¶

Please file an issue on GitHub if you encounter unexpected behavior or if you would like to request new features.

Send a Tweet¶

You can tweet @Ofekmeister anytime.

IRC¶

You find me in Freenode channels #python and #bitcoin as ofek.

Email¶

You can also email me at ofekmeister@gmail.com. Though, if you intend to ask a question that others may benefit from knowing the answer, please use a more public option.
